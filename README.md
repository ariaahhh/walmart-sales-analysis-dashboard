# Walmart Sales Analysis Dashboard

## Project Overview
This project analyzes Walmart weekly sales data to evaluate store performance, sales trends, and external factor impact using Microsoft Excel.

## Dataset
- 6,436 records of weekly sales data
- Features include: Store, Date, Weekly Sales, Holiday Flag, Temperature, Fuel Price, CPI, and Unemployment

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Data Cleaning & Preprocessing
- Data Visualization

## Key KPIs
- Total Sales: ₹6.73 Billion
- Average Weekly Sales: ₹1.04 Million
- Top Performing Store: Store 20

## Insights
- July recorded the highest monthly sales.
- January showed the lowest sales performance.
- Holiday weeks generated higher revenue compared to non-holiday weeks.
- Weak correlation observed between fuel price, temperature, and sales.

## Dashboard Features
- Interactive Store-level slicer
- Monthly Sales Trend Analysis
- Temperature vs Sales Scatter Analysis
- Fuel Price vs Sales Scatter Analysis
- KPI Cards for executive summary
